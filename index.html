Welcome to Mysqto's OpenWrt Mirror.

This is the personal OpenWrt package mirror maintained by @Mysqto. Currently hosted the following packages for ramips(Xiaomi Wifi Mini) based on the OpenWrt official snapshot builds:

Name Description
redsocks redsocks(With OpenSSL) for OpenWRT
openwrt-chinadns ChinaDNS-C for OpenWRT
openwrt-shadowvpn ShadowVPN for OpenWrt
openwrt-dnsmasq Dnsmasq Patch & Makefile for OpenWrt
openwrt-shadowsocks Shadowsocks-libev for OpenWrt
openwrt-dist-luci LuCI Applications for OpenWrt-dist

Uasge

Add the following line to your /etc/opkg.conf file starting with something like src/gz openwrt_dist

http://dist.openwrt.link/releases/ramips/mt7620/packages/base

And run 'opkg update' to update the package source.
